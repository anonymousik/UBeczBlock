Pełna konfiguracja uBlock Origin Lite

Optymalizacja dla YouTube i Chromecast (Gen1/Gen2)

Pobierz i zaimportuj konfigurację

Kliknij poniższy przycisk, aby pobrać plik konfiguracji. Następnie zaimportuj go w rozszerzeniu (Ustawienia → Import / Export).

Instalacja uBlock Origin Lite

  • Chrome / Edge: Chrome Web Store
  • Firefox: Mozilla AMO

Przykładowe filtry

youtube.com##+js(set,y.config_.Config.supportPopupsBlockMessage,false)
youtube.com##+js(set,.prototype.adFound,0)
chromecast.com##+js(aopr,window.navigator.userAgent,'ast_lite_chromecast_gen12')
chromecast.com##+js(set,window.navigator.plugins.length, 1)
